## Overview

The portfolio is designed to function like a desktop environment, where each section opens in its own window.  
This approach helps keep the layout clean while allowing visitors to navigate through different sections naturally.

---

## Features

- Desktop-style interface with draggable and resizable windows  
- Sections for About, Projects, Skills, Certifications, and Contact  
- Dark and light theme support  
- Fully responsive design for desktop and mobile  

---

## Tech Stack

- Next.js 16  
- React 19  
- Tailwind CSS 4  
- Framer Motion  
- shadcn/ui  

---

## Running the Project Locally

# Install dependencies
npm install

# Start the development server
npm run dev

# Build the project for production
npm run build
